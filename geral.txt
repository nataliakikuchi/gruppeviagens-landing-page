chamando as classes: para chamar uma classe colocar ponto e o nome da classe

.header ou div.header = mesma coisa

.xxxxx = tudo o que tem ponto alguma coisa é uma classe

id  = como se fosse uma classe, mas não pode se repetir

* = pega o código todo (mesma fonte, mesma cor)

sem o ponto é uma tag ex.: p

div, p, .intro = com vírgula seleciona todas as div, todos os p e a classe intro

div p = todos os p dentro da div

div > p = pega os p filhos diretos da div

div + p = pega os p irmãos da div que vem imediatamente abaixo (se estiver duas linhas abaixo não pega)

div ~ p = pega os p irmãos que vem imediatamente acima (na mesma linha)

a navegação estruturalmente fica fora do HEADER

display: inline um elemento ao lado do outro na mesma linha. Não considera unidades de tamanho

display: block um elemento abaixo do outro

display: inline-block considera unidades de tamanho

display: flex; com esse display conseguimos mover elementos com mais facilidade, centralizar, por exemplo
só conseguimos usar a propriedade justify-content com o display flex. Ele coloca blocos, um ao lado do outro


padding: espaçamento interior do elemento
margin: afasta os elem entos

position: fixed; manda o elemento para outro plano, normalmente na barra superior de navegação. A deixa congelada


background ou background-image: url(../img/caminhodaimagem); = colocar uma imagem no fundo da página
Se colocarmos uma imagem pequena aqui, por padrão, ela se repete. Se não quisermos repetir, temos que colocar a propriedade

BACKGROUND-REPEAT: com o valor NO-REPEAT

background-position= posiciona a imagemvh onde queremos (eixo Y vertical de cima para baixo e eixo X horizontal da esquerda para a direita) 

background-size: VALOR COVER = pega o tamanho máximo da largura da tela e ajusta a altura. Se ajusta a partir da largura.
Se tirarmos ele deixa de ajustar a imagem e passa a corta-la

min-height= especifica a altura mínima do elemento | valor VH=viewport height (navegador/tela) 100vh= 100% da altura da tela

PX = valor absoluto, estático
